# **App Name**: Org Chart Pro

## Core Features:

- Hierarchical Display: Display the organization in a hierarchical tree structure with Coordinator at the top, followed by Leads and their Members.
- Member Profile: Each member node will display their photo, name, department, year, and role.
- Dynamic Expansion/Collapse: Users can expand and collapse sections of the hierarchy to focus on specific teams.
- AI Org Chart Suggestions: Utilize AI to provide suggestions for re-orgs based on skill sets and group projects in 'tool' usage

## Style Guidelines:

- Primary color: Slate blue (#778DA9) to create a calm, trustworthy, professional impression.
- Background color: Light gray (#F0F0F0) to ensure readability and a clean aesthetic.
- Accent color: Dusty orange (#E29578) for interactive elements and highlighting selected nodes.
- Body and headline font: 'Inter' for a clean, modern, highly readable appearance.
- Use simple, outlined icons to represent departments and roles.
- Utilize a clear, intuitive layout with well-defined sections and consistent spacing to ensure easy navigation.
- Use subtle transitions when expanding/collapsing nodes to provide a smooth user experience.
